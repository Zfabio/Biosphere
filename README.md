## ✨ Features

### 🔬 Interactive Cell Atlas
- **Organelles & Biomolecules** with detailed biological information.
- Smooth 3D visualization of Nuclei, DNA, and Mitochondria.
- Category-based color coding (Nuclear, Energy, Synthesis, etc.).

### 🧬 Genetics Engine
- **Punnett Square Generator** with real-time probability statistics.
- Interactive genotype mapping for Mendelian inheritance.
- Visual feedback for offspring ratios.

### 🧪 Bio Lab Tools
| Tool | Description | Grade Level |
|------|-------------|-------------|
| **Punnett Square** | Predict genetic probability from parental genotypes | 9-12 |
| **DNA Translator** | Translate DNA sequences into RNA and amino acids | 10-12 |
| **Cell Simulator** | Explore organelle functions with interactive models | 9-12 |

## 🚀 Quick Start

### Local Development
```bash
# Clone the repository
git clone https://github.com/Zfabio/biosphere.git

# Navigate to project directory
cd biosphere

# Install dependencies
npm install

# Start dev server (with hot reload)
npm run dev
```

### Quality Checks
```bash
# Lint + syntax check + production build
npm run check
```

## 🎨 Design Philosophy

BioSphere follows modern organic design principles:

- **Organic & Clean** - Soft emerald and teal palettes.
- **Glassmorphism** - Premium frosted glass UI elements.
- **Responsive** - Optimized for desktop and mobile learning.
- **Micro-animations** - Fluid transitions for 3D biological structures.
